## Installation ‚ö°

Follow these steps to run HackDevs locally:

1. **Clone the repository**

```bash
git clone https://github.com/YourUsername/HackDevs.git
cd Hackathon
```
2. **Install depedencies**

```
npm i
```
2. **Run the Project**
```
npm run dev
```
